# **App Name**: Cabanatuan Boarder Finder

## Core Features:

- User Authentication: User onboarding modal with signup/login (email/password or Google OAuth).
- User Preference Survey: Post-authentication survey to capture user preferences (student/worker, accommodation type, price range, amenities, and location).
- Property Listings Display: Display boarding houses in a responsive grid, each card showing property details.
- Property Detail View: Display a detail page when a listing card is clicked, with a photo gallery.
- Search and Filtering: Implement filtering of boarding houses using price, accommodation type, amenities and distance.
- Owner Contact and Messaging: Add contact info and chat to allow user to directly connect with the owner.

## Style Guidelines:

- Primary color: A warm, inviting yellow (#FFC107) to evoke a sense of home and comfort.
- Background color: A light, desaturated yellow (#FAF8EE), for a calm and clean foundation.
- Accent color: A complementary orange (#FF9800) for interactive elements and calls to action, suggesting excitement and discovery.
- Body and headline font: 'PT Sans' (sans-serif) provides a modern and approachable feel for all text elements.
- Simple, clear icons to represent amenities and property features.
- Clean and organized grid layout for property listings.
- Subtle transitions and animations to enhance user interaction.
